1) Create a new Azure WebApp with SQL Server in a single resource group
2) Connect to SQL Server DB with VS and run the create table script to create Students table
3) Run script to populate the table with test data.
4) Open up VS Online code editor in Azure portal and upload the index.cshtml file and web.config files
5) Edit the SQL Connections string in index.html. This currently assumes one called "DefaultConnection"
6) Hit the web page and see students list
